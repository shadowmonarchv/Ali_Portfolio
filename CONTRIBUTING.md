# Contributing to Ali Mehdi Mirza's Portfolio

Thanks for your interest in contributing! Here's how to get started.

## Getting Started
1. Fork this repository
2. Clone your forked repo locally
```bash
   git clone https://github.com/YOUR_USERNAME/Ali_Portfolio.git
   cd Ali_Portfolio
```
3. Create a new branch for your fix
```bash
   git checkout -b fix/issue-description
```
4. Make your changes
5. Commit with a descriptive message
```bash
   git commit -m "feat: description of what you did - closes #ISSUE_NUMBER"
```
6. Push to your fork
```bash
   git push origin fix/issue-description
```
7. Open a Pull Request to this repo's `main` branch

## Rules
- One issue per pull request only
- Always link the issue in your PR description using `Closes #NUMBER`
- Test your changes on mobile and desktop before submitting
- Follow the existing code style — no tabs, use 2 spaces
- Do not change the color scheme or typography

## Questions
Open a new issue with the `question` label if you need help.
